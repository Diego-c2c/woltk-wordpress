# WoTLK WordPress Pack

Pack prêt à pousser sur GitHub puis à cloner sur le VPS.

## Contenu
- `docker-compose.yml`
- `theme/woltk-portal/`
- `docs/INSTALLATION.md`

## Déploiement rapide
```bash
cd ~/site
git clone https://github.com/Diego-c2c/woltk-wordpress.git
cd woltk-wordpress
docker compose up -d
```
